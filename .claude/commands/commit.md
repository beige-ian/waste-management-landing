# 변경사항 저장 및 푸시

작업한 내용을 GitHub에 저장하세요:

```bash
# 1. 변경사항 확인
git status

# 2. 모든 파일 추가
git add .

# 3. 커밋
git commit -m "작업 내용 설명"

# 4. GitHub에 푸시
git push
```

---

### 한 줄로 실행:
```bash
git add . && git commit -m "작업 내용" && git push
```

---

### 커밋 메시지 예시:
- `git commit -m "Fix: 건물주의 현실 섹션 UI 개선"`
- `git commit -m "Feature: 새로운 컴포넌트 추가"`
- `git commit -m "Style: 색상 및 폰트 조정"`
- `git commit -m "Refactor: ProblemSolution 컴포넌트 정리"`
