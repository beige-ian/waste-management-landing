# 개발 서버 시작

개발 모드로 프로젝트를 실행하세요:

```bash
npm run dev
```

---

### 화면에 나타나는 내용:
```
  VITE v5.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  press h to show help
```

**http://localhost:5173** 에서 프로젝트를 확인할 수 있습니다.

---

### 파일 저장 시 자동 새로고침
- 파일을 저장하면 자동으로 브라우저에서 새로고침됩니다 (HMR)
- CSS, TypeScript, React 파일 모두 지원

### 개발 서버 종료

```bash
Ctrl + C
```
